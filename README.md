# Student Score Predictor

## 📌 Project Description
This project is a python-based student score predictor that predicts the score of students on the basis of given number of hours studied.

## 🚀 Features

- Predicts Score
- Hours studied vs score graph
- Slope of graph
- Intercept of graph

## 🛠  Technologies Used

- Python
- Pandas
- Numpy
- Matplotlib

## 📦 Installation

make sure you have Python installed.

pip install pandas
pip install numpy
pip install matplotlib

## ▶️ How to Run

1. Clone the repository:
    git clone https://github.com/shivngi29/Student-Score-Prediction.git

2. Navigate to project folder:
    cd StudentScorePrediction

3. Run the program:
    python StudentScorePrediction.py
